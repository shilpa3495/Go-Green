# Go-Green

Component Library

## General Info

GoGreen is a simple, modular and responsive component library that have build on plain HTML, CSS and JS. It will also help to build your project fast.

## Technologies
Project is created with:
* HTML
* CSS
* JS

## Features
* Responsive
* Dark and Light mode
* Landing page
